可存储任意类型对象，可变。字典包含在花括号里面，键值对用冒号分割，对之间用逗号分割，键key必须是唯一的，不可变的，如字符串，数字，元组。

字典内置函数方法

len（）计算字典元素数

str（dict）输出字典以字符串形式

type（var）返回输入的变量类型

dict.clear（）清空字典

dict.copy（）返回字典浅复制

dict.values（）以列表返回字典所有值

dict.key（）类似上

dict.update（dict2）把dict2键值对更新进dict中

dict.get（key，default=）返回指定key的值，如果没有返回default的值

dict.setdefault（key，default=）与get类似，只是若key不存在，则加进去键值对

key in dict 返回布尔值

dict.items（）以列表返回可遍历的键值元组数组
